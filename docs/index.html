🚀 JavaFX & Collections: Practical Guide with Examples

1. JavaFX Core Interface Elements

JavaFX provides the structural components and interactive widgets needed to build graphical applications [1, 2].

ElementExplanation & Practical Use
Application, launch(), & start() A JavaFX application must inherit from the Application class [1]. The launch() function starts the app, which then calls the start(Stage) method where you set up your initial window [1, 3, 4].
public class Main extends Application { public static void main(String[] args) { launch(args); // Boots the app } @Override public void start(Stage primaryStage) { // App setup goes here } }
Stage & Scene The Stage is your main application window, and the Scene is the background container holding all your visual components [1, 2]. You attach the Scene to the Stage to make elements visible [5].
Scene scene = new Scene(root, 500, 300); primaryStage.setScene(scene); primaryStage.setTitle("LOGIN"); primaryStage.show();
Label, TextField & PasswordField Label displays static text, TextField accepts normal text input, and PasswordField hides the text typed by the user [6, 7].
Label title = new Label("Search"); TextField username = new TextField(); PasswordField pf = new PasswordField(); pf.setPromptText("Your password");
Button & ComboBox A Button triggers actions when clicked, and can contain text or images [8]. A ComboBox allows users to select one option from a dropdown list [9].
Button acceptBtn = new Button("Accept"); ComboBox combo = new ComboBox(); combo.getItems().addAll("Option 1", "Option 2");
Layouts:
BorderPane & HBox		 Layouts automatically organize widgets. BorderPane places elements in 5 zones (Top, Bottom, Left, Right, Center) [10]. HBox lines up elements horizontally in a row [10, 11].
// BorderPane Example BorderPane bp = new BorderPane(); bp.setCenter(new Button("centre")); // HBox Example HBox hbox = new HBox(); hbox.setPadding(new Insets(15, 12, 15, 12)); hbox.setSpacing(10);
Alert (Dialog Boxes) Used to pop up a separate dialog window to notify the user, show warnings, or confirm actions [12].
Alert alert = new Alert(AlertType.INFORMATION); alert.setTitle("Test Connection"); alert.setHeaderText("Results:"); alert.setContentText("Connected successfully!"); alert.showAndWait();

2. Java Generics & Collections

The Java Collections Framework provides standard data structures like Lists and Maps, and Generics (<T>) ensure that your data stays strictly organized by type [13, 14].

ElementExplanation & Practical Use
ArrayList<E> A dynamic array that resizes itself automatically. Unlike standard arrays, they are highly recommended because they strictly enforce the data type using Generics (e.g., <Integer>), preventing crashes [13-15].
ArrayList<Integer> listeEntiers = new ArrayList<>(); listeEntiers.add(56); Integer x = listeEntiers.get(0); // Retrieves 56
Map<K, V> & HashMap Works like a dictionary, mapping a unique "Key" (K) to a "Value" (V) [14, 16]. Great for looking up data quickly without looping.
Map<String, Integer> villes = new HashMap<>(); villes.put("Paris", 75); // Add key-value villes.put("Montpellier", 34); // Retrieve value using its key Integer zip = villes.get("Montpellier"); // Gets 34
Comparable<T>, compareTo() & Collections.sort() To sort custom objects automatically (like Students or Cities), the object's class must implement Comparable and define a compareTo() function [17, 18]. Once set up, you can sort entire lists with a single command [19].
public class Eleve implements Comparable<Eleve> { public int moyenne; // Define how to compare two students public int compareTo(Eleve autre) { if (this.moyenne < autre.moyenne) return -1; if (this.moyenne > autre.moyenne) return 1; return 0; } } // In your main code: List<Eleve> le = new ArrayList<>(); Collections.sort(le); // Automatically sorts the list
